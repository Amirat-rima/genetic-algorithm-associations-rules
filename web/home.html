Le probleme d'extraction des règles d'association

Dans le domaine du data mining la recherche des règles d'association est une méthode populaire permet de découvrir des relations ayant un intérêt pour le statisticien entre deux ou plusieurs variables stockées dans les bases de données. Par exemple, une règle découverte dans les données de ventes dans un supermarché pourrait indiquer qu'un client achetant du lait et du beurre simultanément, serait susceptible d'acheter du pain. Une telle information peut être utilisée comme base pour prendre des décisions marketing.
